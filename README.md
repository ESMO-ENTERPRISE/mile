# MILEFU
Build any Typescript or Javascript project with minimum configuration.

### How to use?
1. First install milefu cli :
   ```bash
    npm i -g @vira/milefu
    ```
2. Initialize project:
    ```bash
    milefu -i
    ```
    OR
    ```bash
    milefu --init
    ```
3. Build the project:
    ```bash
    milefy -b
    ```
    OR
    ```bash
    milefy --build
    ```
